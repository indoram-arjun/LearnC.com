Arrays in C Programming

An array stores multiple values of the same data type under one name. Arrays are useful when you need to work with lists, marks, tables, matrices, and repeated data.

Loops Arrays Strings Functions

What is an Array?

An array is a collection of values stored in continuous memory locations. All values must have the same data type, such as all int values or all float values.

  • Array elements are accessed using an index.
  • Index starts from 0, not 1.
  • If an array has 5 elements, valid indexes are 0 to 4.
Simple idea: Array ko ek row of boxes samjho. Har box me same type ka data hota hai, aur box number index ke through access hota hai.

Declaring and Initializing Arrays

Syntax Meaning
int marks[5]; Declares an integer array with 5 elements.
int nums[5] = {10, 20, 30, 40, 50}; Declares and initializes all 5 elements.
int nums[] = {2, 4, 6}; Compiler automatically counts the size as 3.
array_basic.c 
#include <stdio.h>

int main() {
    int marks[5] = {78, 85, 91, 64, 72};

    printf("First mark = %d\n", marks[0]);
    printf("Third mark = %d\n", marks[2]);

    marks[3] = 70;   // update fourth element
    printf("Updated fourth mark = %d\n", marks[3]);

    return 0;
}
// Output

First mark = 78
Third mark = 91
Updated fourth mark = 70

Traversing an Array

Traversing means visiting every element of an array. Loops and arrays are usually used together because indexes follow a simple numeric pattern.

array_traversal.c 
#include <stdio.h>

int main() {
    int nums[5] = {10, 20, 30, 40, 50};
    int sum = 0;

    for (int i = 0; i < 5; i++) {
        printf("nums[%d] = %d\n", i, nums[i]);
        sum = sum + nums[i];
    }

    printf("Sum = %d\n", sum);
    return 0;
}
// Output

nums[0] = 10
nums[1] = 20
nums[2] = 30
nums[3] = 40
nums[4] = 50
Sum = 150

Input and Output with Arrays

You can use scanf() with a loop to input many values. Remember to pass the address of each element using &arr[i].

array_input.c 
#include <stdio.h>

int main() {
    int marks[3];

    for (int i = 0; i < 3; i++) {
        printf("Enter mark %d: ", i + 1);
        scanf("%d", &marks[i]);
    }

    printf("You entered: ");
    for (int i = 0; i < 3; i++) {
        printf("%d ", marks[i]);
    }

    return 0;
}
Common mistake: Do not access outside the array size. For marks[3], indexes are only 0, 1, 2.

Searching in an Array

Linear search checks elements one by one until the required value is found or the array ends.

linear_search.c 
#include <stdio.h>

int main() {
    int nums[5] = {12, 45, 23, 67, 34};
    int key = 23;
    int found = 0;

    for (int i = 0; i < 5; i++) {
        if (nums[i] == key) {
            printf("Found at index %d\n", i);
            found = 1;
            break;
        }
    }

    if (found == 0) {
        printf("Not found\n");
    }

    return 0;
}
// Output

Found at index 2

Sorting an Array

Sorting means arranging array elements in a specific order. This example uses bubble sort to arrange values in ascending order.

bubble_sort.c 
#include <stdio.h>

int main() {
    int a[5] = {30, 10, 50, 20, 40};
    int temp;

    for (int i = 0; i < 4; i++) {
        for (int j = 0; j < 4 - i; j++) {
            if (a[j] > a[j + 1]) {
                temp = a[j];
                a[j] = a[j + 1];
                a[j + 1] = temp;
            }
        }
    }

    printf("Sorted array: ");
    for (int i = 0; i < 5; i++) {
        printf("%d ", a[i]);
    }

    return 0;
}
// Output

Sorted array: 10 20 30 40 50

2D Arrays

A 2D array stores data in rows and columns. It is useful for matrices, tables, marksheets, seating plans, and grid-based data.

matrix.c 
#include <stdio.h>

int main() {
    int matrix[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };

    for (int row = 0; row < 2; row++) {
        for (int col = 0; col < 3; col++) {
            printf("%d ", matrix[row][col]);
        }
        printf("\n");
    }

    return 0;
}
// Output

1 2 3
4 5 6
