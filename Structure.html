Structures in C Programming

A structure is a user-defined data type that groups related variables of different types under one name — like a student record with name, age, and marks.

What is a Structure?

A structure (struct) is a user-defined data type that allows you to combine data items of different types. For example, a Student has a name (string), age (int), and marks (float) — all grouped together in one struct.

  • Structures allow grouping of related data of different types.
  • Declared using the struct keyword.
  • Members are accessed using the dot operator (.).
  • Pointers to structs use the arrow operator (->).

Declaring and Using a Structure

struct_basic.c 
#include <stdio.h>
#include <string.h>

// Define the structure
struct Student {
    char name[50];
    int age;
    float marks;
};

int main() {
    // Declare a structure variable
    struct Student s1;

    // Assign values
    strcpy(s1.name, "Riya");
    s1.age = 20;
    s1.marks = 92.5;

    // Print values
    printf("Name  : %s\n", s1.name);
    printf("Age   : %d\n", s1.age);
    printf("Marks : %.1f\n", s1.marks);

    return 0;
}
// Output

Name : Riya
Age : 20
Marks : 92.5

Array of Structures

We can create multiple records using an array of structures — very useful for storing student lists, employee records, etc.

struct_array.c 
#include <stdio.h>
#include <string.h>

struct Student {
    char name[50];
    int roll;
    float marks;
};

int main() {
    struct Student class[3] = {
        {"Aarav",  1, 88.0},
        {"Priya",  2, 95.5},
        {"Rohan",  3, 76.0}
    };

    printf("%-10s %-6s %-6s\n", "Name", "Roll", "Marks");
    printf("-----------------------------\n");
    for (int i = 0; i < 3; i++) {
        printf("%-10s %-6d %.1f\n",
               class[i].name, class[i].roll, class[i].marks);
    }
    return 0;
}
// Output

Name Roll Marks
-----------------------------
Aarav 1 88.0
Priya 2 95.5
Rohan 3 76.0

Practice Programs

1. Structure to store and display book information
program.c 
#include <stdio.h>
#include <string.h>
struct Book {
    char title[100];
    char author[50];
    int year;
    float price;
};
int main() {
    struct Book b = {"C Programming Language", "Kernighan & Ritchie", 1978, 450.0};
    printf("Title  : %s\n", b.title);
    printf("Author : %s\n", b.author);
    printf("Year   : %d\n", b.year);
    printf("Price  : Rs.%.2f\n", b.price);
    return 0;
}
// Output

Title : C Programming Language
Author : Kernighan & Ritchie
Year : 1978
Price : Rs.450.00

2. Find the student with highest marks
program.c 
#include <stdio.h>
struct Student { char name[30]; float marks; };
int main() {
    struct Student s[3] = {{"Alice",88},{"Bob",95},{"Carol",79}};
    int top = 0;
    for (int i = 1; i < 3; i++)
        if (s[i].marks > s[top].marks) top = i;
    printf("Topper: %s (%.1f)\n", s[top].name, s[top].marks);
    return 0;
}
// Output

Topper: Bob (95.0)
