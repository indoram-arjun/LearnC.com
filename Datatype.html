Data Types in C Programming

Data types define the type and size of data a variable can hold. Every variable in C must be declared with a data type before use.

What are Data Types?

A data type tells the compiler what kind of data a variable will store — whether it's a number, a decimal, or a character. It also determines how much memory is allocated.

  • Data types define the size and range of values.
  • The compiler uses the data type to decide how many bytes to reserve in memory.
  • C is a statically typed language — you must declare the type before using a variable.

Primary (Basic) Data Types

C has four fundamental primary data types:

Data Type Size Range Format Specifier
int 4 bytes -2,147,483,648 to 2,147,483,647 %d
float 4 bytes 3.4e-38 to 3.4e+38 %f
double 8 bytes 1.7e-308 to 1.7e+308 %lf
char 1 byte -128 to 127 %c

Using Basic Data Types — Example

datatypes.c 
#include <stdio.h>

int main() {
    int age = 20;
    float height = 5.9;
    double pi = 3.14159265358979;
    char grade = 'A';

    printf("Age     : %d\n", age);
    printf("Height  : %.1f\n", height);
    printf("Pi      : %.10lf\n", pi);
    printf("Grade   : %c\n", grade);

    // Check memory sizes
    printf("\nSizes:\n");
    printf("int    = %lu bytes\n", sizeof(int));
    printf("float  = %lu bytes\n", sizeof(float));
    printf("double = %lu bytes\n", sizeof(double));
    printf("char   = %lu bytes\n", sizeof(char));

    return 0;
}
// Output

Age : 20
Height : 5.9
Pi : 3.1415926536
Grade : A

Sizes:
int = 4 bytes
float = 4 bytes
double = 8 bytes
char = 1 byte

Derived Data Types

Derived data types are built from primary types:

Array

A collection of elements of the same type stored in contiguous memory. int arr[5];

Pointer

Stores the memory address of another variable. int *p;

Function

A block of code that performs a specific task and can return a value.

Structure

Groups different data types under one name. struct Student { };

Type Modifiers

Modifiers change the size or sign of a data type:

Modified Type Size Range
short int 2 bytes -32,768 to 32,767
long int 8 bytes -9.2×10¹⁸ to 9.2×10¹⁸
unsigned int 4 bytes 0 to 4,294,967,295
unsigned char 1 byte 0 to 255
long double 12–16 bytes Higher precision than double

Type Conversion

C allows converting one data type to another — either automatically (implicit) or manually (explicit).

typecast.c 
#include <stdio.h>

int main() {
    // Implicit conversion (automatic)
    int a = 5;
    float b = a;     // int automatically becomes float
    printf("b = %.1f\n", b);  // Output: 5.0

    // Explicit conversion (type casting)
    float x = 9.7;
    int y = (int)x;  // Decimal part is lost
    printf("y = %d\n", y);    // Output: 9

    // Integer division vs float division
    int p = 7, q = 2;
    printf("int  : %d\n", p / q);         // Output: 3
    printf("float: %.1f\n", (float)p / q); // Output: 3.5

    return 0;
}
// Output

b = 5.0
y = 9
int : 3
float: 3.5

Practice Programs

1. Print sizes of all data types
sizes.c 
#include <stdio.h>
int main() {
    printf("char         = %lu bytes\n", sizeof(char));
    printf("int          = %lu bytes\n", sizeof(int));
    printf("short int    = %lu bytes\n", sizeof(short int));
    printf("long int     = %lu bytes\n", sizeof(long int));
    printf("float        = %lu bytes\n", sizeof(float));
    printf("double       = %lu bytes\n", sizeof(double));
    printf("long double  = %lu bytes\n", sizeof(long double));
    return 0;
}
// Output

char = 1 bytes
int = 4 bytes
short int = 2 bytes
long int = 8 bytes
float = 4 bytes
double = 8 bytes
long double = 12 bytes

2. Demonstrate type casting in division
casting.c 
#include <stdio.h>
int main() {
    int a = 10, b = 3;
    printf("Without cast : %d\n", a / b);
    printf("With cast    : %.4f\n", (float)a / b);
    printf("With cast    : %.4lf\n", (double)a / b);
    return 0;
}
// Output

Without cast : 3
With cast : 3.3333
With cast : 3.3333

3. Store and display char as integer
ascii.c 
#include <stdio.h>
int main() {
    char ch = 'A';
    printf("Character : %c\n", ch);
    printf("ASCII     : %d\n", ch);
    printf("Next char : %c\n", ch + 1);  // 'B'
    return 0;
}
// Output

Character : A
ASCII : 65
Next char : B
