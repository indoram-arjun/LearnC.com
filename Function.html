Functions in C Programming

A function is a reusable block of code that performs one clear task. Functions make large programs easier to read, test, and reuse.

Strings Functions Structures

What is a Function?

A function is a named block of statements. Instead of writing the same code again and again, you write it once inside a function and call it whenever needed.

Reusable Use the same logic many times.
Readable Split a big program into small parts.
Testable Check one task at a time.
Easy idea: Function ek chhota machine jaisa hota hai - input do, kaam karata hai, aur optional output return karta hai.

Function Parts

Part Meaning Example
Declaration Tells the compiler that a function exists. int add(int, int);
Call Runs the function from another place. sum = add(4, 6);
Definition Contains the actual function body. int add(int a, int b) { ... }

Basic Function Example

This program defines a function named add and calls it from main().

add_function.c 
#include <stdio.h>

int add(int a, int b);   // function declaration

int main() {
    int result;

    result = add(10, 20);    // function call
    printf("Sum = %d\n", result);

    return 0;
}

int add(int a, int b) {  // function definition
    return a + b;
}
// Output

Sum = 30

Types of User-defined Functions

Type Example Function Use Case
No return, no arguments void greet(void) Print a fixed message.
No return, with arguments void printSquare(int n) Use input but only print result.
Return, no arguments int getNumber(void) Calculate or read a value and return it.
Return, with arguments int max(int a, int b) Most common: input goes in, result comes back.

Actual and Formal Parameters

Actual parameters are the values sent during a function call. Formal parameters are the variables that receive those values inside the function.

parameters.c 
#include <stdio.h>

void showMarks(int marks) {       // marks is a formal parameter
    printf("Marks = %d\n", marks);
}

int main() {
    int studentMarks = 86;

    showMarks(studentMarks);      // studentMarks is an actual parameter

    return 0;
}
// Output

Marks = 86

Return Value

The return statement sends a value back to the caller and ends the function. If a function has return type void, it does not return a value.

largest.c 
#include <stdio.h>

int largest(int a, int b) {
    if (a > b) {
        return a;
    }
    return b;
}

int main() {
    printf("Largest = %d\n", largest(45, 31));
    return 0;
}
// Output

Largest = 45

Recursion

When a function calls itself, it is called recursion. Every recursive function needs a stopping condition; otherwise it keeps calling itself forever.

factorial.c 
#include <stdio.h>

int factorial(int n) {
    if (n == 0 || n == 1) {
        return 1;      // stopping condition
    }
    return n * factorial(n - 1);
}

int main() {
    printf("Factorial = %d\n", factorial(5));
    return 0;
}
// Output

Factorial = 120

Remember: Recursion mein base condition compulsory hai. Base condition nahi hogi to function stop nahi karega.
