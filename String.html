Strings in C Programming

A string in C is an array of characters terminated by a null character '\0'. C provides a rich library of string functions in <string.h>.

What is a String?

In C, a string is a sequence of characters stored in a character array. It always ends with a special character called the null terminator ('\0').

  • Strings in C are arrays of type char.
  • The null character '\0' marks the end of the string.
  • String functions are available in the <string.h> header.
string_decl.c 
#include <stdio.h>

int main() {
    // Method 1: Character array with size
    char name[20] = "Nikita";

    // Method 2: Without size (compiler decides)
    char city[] = "Delhi";

    // Method 3: Pointer to string literal
    char *msg = "Hello, C!";

    printf("Name: %s\n", name);
    printf("City: %s\n", city);
    printf("Message: %s\n", msg);
    return 0;
}
// Output

Name: Nikita
City: Delhi
Message: Hello, C!

String Functions — <string.h>

Function Description Example
strlen(s) Returns length of string (excluding \0) strlen("hello") → 5
strcpy(d, s) Copies string s into d strcpy(dest, "hi")
strcat(d, s) Appends string s to end of d strcat(a, b)
strcmp(s1, s2) Compares two strings (0 = equal) strcmp("a","a") → 0
strupr(s) Converts string to uppercase strupr("hello") → HELLO
strlwr(s) Converts string to lowercase strlwr("HI") → hi
strrev(s) Reverses the string strrev("abc") → cba

String Function Examples

string_funcs.c 
#include <stdio.h>
#include <string.h>

int main() {
    char s1[50] = "Hello";
    char s2[] = " World";
    char s3[50];

    // strlen
    printf("Length: %lu\n", strlen(s1));

    // strcat
    strcat(s1, s2);
    printf("Concatenated: %s\n", s1);

    // strcpy
    strcpy(s3, "C Programming");
    printf("Copied: %s\n", s3);

    // strcmp
    int result = strcmp("apple", "apple");
    printf("Compare: %d\n", result);  // 0 = equal

    return 0;
}
// Output

Length: 5
Concatenated: Hello World
Copied: C Programming
Compare: 0

Practice Programs

1. Count vowels in a string
program.c 
#include <stdio.h>
#include <string.h>
int main() {
    char str[] = "Hello World";
    int count = 0;
    for (int i = 0; str[i] != '\0'; i++) {
        char c = str[i];
        if(c=='a'||c=='e'||c=='i'||c=='o'||c=='u'||
           c=='A'||c=='E'||c=='I'||c=='O'||c=='U')
            count++;
    }
    printf("Vowels: %d\n", count);
    return 0;
}
// Output

Vowels: 3

2. Reverse a string without strrev
program.c 
#include <stdio.h>
#include <string.h>
int main() {
    char str[] = "Programming";
    int n = strlen(str);
    for (int i = 0; i < n / 2; i++) {
        char temp = str[i];
        str[i] = str[n - 1 - i];
        str[n - 1 - i] = temp;
    }
    printf("Reversed: %s\n", str);
    return 0;
}
// Output

Reversed: gnimmargorP
