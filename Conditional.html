Conditional Statements in C Programming

Conditional statements allow a program to make decisions and execute different code blocks based on whether a condition is true or false.

What are Conditional Statements?

A conditional statement controls the flow of a program by executing a block of code only when a certain condition is met. Without them, programs would just run from top to bottom with no decisions.

  • C evaluates conditions as true (non-zero) or false (0).
  • Conditional statements use relational and logical operators.
  • Types: if, if-else, if-else-if, nested if, switch.

if Statement

Executes the block only when the condition is true. Syntax: if (condition) { }

if_stmt.c 
#include <stdio.h>

int main() {
    int marks = 75;

    if (marks >= 40) {
        printf("You have passed!\n");
    }

    if (marks >= 90) {
        printf("Distinction!\n");  // This won't print
    }

    printf("Program continues...\n");
    return 0;
}
// Output

You have passed!
Program continues...

if-else Statement

Executes one block if the condition is true, another if it's false. Syntax: if (condition) { } else { }

if_else.c 
#include <stdio.h>

int main() {
    int num = -5;

    if (num > 0) {
        printf("%d is Positive\n", num);
    } else {
        printf("%d is Negative or Zero\n", num);
    }

    // Check even or odd
    int n = 14;
    if (n % 2 == 0) {
        printf("%d is Even\n", n);
    } else {
        printf("%d is Odd\n", n);
    }

    return 0;
}
// Output

-5 is Negative or Zero
14 is Even

if-else-if Ladder

Used to check multiple conditions one by one. The first true condition's block executes, then the rest are skipped.

ladder.c 
#include <stdio.h>

int main() {
    int marks = 82;
    char grade;

    if (marks >= 90) {
        grade = 'A';
    } else if (marks >= 80) {
        grade = 'B';
    } else if (marks >= 70) {
        grade = 'C';
    } else if (marks >= 60) {
        grade = 'D';
    } else {
        grade = 'F';
    }

    printf("Marks: %d => Grade: %c\n", marks, grade);
    return 0;
}
// Output

Marks: 82 => Grade: B

Nested if

An if inside another if. Used to check multiple related conditions.

nested_if.c 
#include <stdio.h>

int main() {
    int a = 10, b = 20, c = 15;
    int largest;

    if (a > b) {
        if (a > c) {
            largest = a;
        } else {
            largest = c;
        }
    } else {
        if (b > c) {
            largest = b;
        } else {
            largest = c;
        }
    }

    printf("Largest = %d\n", largest);
    return 0;
}
// Output

Largest = 20

switch-case Statement

Efficiently handles multiple fixed values of a variable. Cleaner than a long if-else-if ladder when checking one variable against constants. Always use break to prevent fall-through.

switch.c 
#include <stdio.h>

int main() {
    int day = 3;

    switch (day) {
        case 1:
            printf("Monday\n");
            break;
        case 2:
            printf("Tuesday\n");
            break;
        case 3:
            printf("Wednesday\n");
            break;
        case 4:
            printf("Thursday\n");
            break;
        case 5:
            printf("Friday\n");
            break;
        default:
            printf("Weekend\n");
    }

    // switch with char
    char op = '+';
    int x = 10, y = 5;
    switch (op) {
        case '+': printf("Sum = %d\n", x + y); break;
        case '-': printf("Diff = %d\n", x - y); break;
        case '*': printf("Prod = %d\n", x * y); break;
        case '/': printf("Div = %d\n", x / y); break;
        default: printf("Unknown operator\n");
    }

    return 0;
}
// Output

Wednesday
Sum = 15

if-else vs switch — When to Use?

Feature if-else switch
Condition type Any expression (ranges, logical) Only integer/char constants
Range check ✅ Supported ❌ Not supported
Readability OK for 2-3 conditions Better for many fixed values
Fall-through No Yes (needs break)

Practice Programs

1. Check if a year is a leap year
leap.c 
#include <stdio.h>
int main() {
    int year = 2024;
    if ((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)) {
        printf("%d is a Leap Year\n", year);
    } else {
        printf("%d is NOT a Leap Year\n", year);
    }
    return 0;
}
// Output

2024 is a Leap Year

2. Find largest of three numbers
largest.c 
#include <stdio.h>
int main() {
    int a = 30, b = 55, c = 45;
    if (a >= b && a >= c)
        printf("Largest: %d\n", a);
    else if (b >= a && b >= c)
        printf("Largest: %d\n", b);
    else
        printf("Largest: %d\n", c);
    return 0;
}
// Output

Largest: 55

3. Simple calculator using switch
calc.c 
#include <stdio.h>
int main() {
    float a = 20, b = 4;
    char op = '/';
    switch (op) {
        case '+': printf("%.2f\n", a + b); break;
        case '-': printf("%.2f\n", a - b); break;
        case '*': printf("%.2f\n", a * b); break;
        case '/':
            if (b != 0) printf("%.2f\n", a / b);
            else printf("Division by zero!\n");
            break;
        default: printf("Invalid operator\n");
    }
    return 0;
}
// Output
