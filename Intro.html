Introduction to C Programming

C is a general-purpose, procedural programming language developed in the early 1970s. It remains one of the most widely used and influential languages ever created.

Introduction Variables Operators Data Types

What is C?

C is a general-purpose, structured programming language developed by Dennis Ritchie at Bell Laboratories in 1972. It was designed to write the UNIX operating system and is considered the "mother of all programming languages."

  • C is a compiled language — code is converted directly to machine code.
  • It gives the programmer low-level access to memory using pointers.
  • C is fast, portable, and efficient — used in OS, embedded systems, and compilers.
  • Languages like C++, Java, Python, and PHP are all influenced by C.

History of C

Year Event
1967 BCPL language created by Martin Richards
1970 Ken Thompson created B language (based on BCPL)
1972 Dennis Ritchie developed C at Bell Labs (based on B)
1978 "The C Programming Language" book published (Kernighan & Ritchie)
1989 ANSI C standard (C89/C90) released
1999 C99 standard with new features
2011 C11 — current widely-used standard

Structure of a C Program

Every C program follows a fixed structure. Understanding this is the first step.

structure.c 
#include <stdio.h>    // Preprocessor directive

// Global variables (optional)

int main() {          // Main function — execution starts here
    // Local variables
    // Statements

    printf("Hello, World!\n");   // Output statement

    return 0;         // Return value to OS
}
// Output

Hello, World!

Tip: int main() is the entry point of every C program. Execution always starts from here.

How C Code is Compiled

Step 1: Write

Write your C code in a .c file using any text editor or IDE.

Step 2: Preprocess

Preprocessor handles #include, #define directives before compilation.

Step 3: Compile

Compiler (GCC) converts C code to assembly/object code (.obj file).

Step 4: Link

Linker combines object files and libraries into an executable (.exe).

Step 5: Run

Execute the .exe file. The OS runs the machine code directly.

Your First C Program

hello.c 
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    printf("Welcome to C Programming!\n");
    return 0;
}
// Output

Hello, World!
Welcome to C Programming!

Features of C

Simple & Efficient

Easy to learn syntax, yet produces very fast and efficient programs.

Portable

C programs written on one machine can run on another with minimal changes.

Rich Library

Large standard library with functions for I/O, strings, math, memory.

Structured

Supports functions, loops, conditionals — encourages organized code.

Low-Level Access

Direct memory manipulation through pointers — great for systems programming.

Extensible

You can add your own functions to the C library and extend it.

Note: C uses printf() for output and scanf() for input. Both are defined in <stdio.h>.
