Operators in C Programming

An operator is a symbol that tells the compiler to perform specific mathematical or logical operations. C is rich in built-in operators.

Arithmetic Operators

Operator Name Example Result
+ Addition 5 + 3 8
- Subtraction 5 - 3 2
* Multiplication 5 * 3 15
/ Division 10 / 3 3
% Modulus 10 % 3 1
++ Increment a++ a = a+1
-- Decrement a-- a = a-1

Relational Operators

Operator Meaning Example
== Equal to a == b
!= Not equal to a != b
> Greater than a > b
< Less than a < b
>= Greater than or equal a >= b
<= Less than or equal a <= b

Logical Operators

Operator Name Description Example
&& AND True if both operands are true a>0 && b>0
|| OR True if at least one is true a>0 || b>0
! NOT Reverses the logical state !flag

Operators Example

operators.c 
#include <stdio.h>

int main() {
    int a = 10, b = 3;

    // Arithmetic
    printf("a + b = %d\n", a + b);
    printf("a %% b = %d\n", a % b);   // modulus
    printf("a / b = %d\n", a / b);

    // Relational
    printf("a > b: %d\n", a > b);   // 1 = true
    printf("a == b: %d\n", a == b);  // 0 = false

    // Logical
    printf("(a>0 && b>0): %d\n", (a > 0 && b > 0));
    printf("(a>0 || b<0): %d\n", (a > 0 || b < 0));

    // Ternary
    int max = (a > b) ? a : b;
    printf("Max = %d\n", max);

    return 0;
}
// Output

a + b = 13
a % b = 1
a / b = 3
a > b: 1
a == b: 0
(a>0 && b>0): 1
Max = 10
