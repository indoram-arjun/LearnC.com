Variables in C Programming

A variable is a named memory location that stores data. It is the most fundamental concept in any programming language.

What is a Variable?

A variable is a container that holds data which can change during program execution. Every variable has a name, a data type, and a value.

  • Variables are stored in RAM memory during program execution.
  • Each variable has a unique memory address.
  • The value of a variable can change but its type cannot.

Declaring Variables

Syntax: data_type variable_name; or data_type variable_name = value;

variables.c 
#include <stdio.h>

int main() {
    int age = 20;           // Integer variable
    float marks = 95.5;     // Float variable
    char grade = 'A';       // Character variable
    double pi = 3.14159;    // Double variable

    printf("Age: %d\n", age);
    printf("Marks: %.1f\n", marks);
    printf("Grade: %c\n", grade);
    printf("Pi: %lf\n", pi);

    return 0;
}
// Output

Age: 20
Marks: 95.5
Grade: A
Pi: 3.141590

Rules for Naming Variables

Rule Example
Must start with a letter or underscore (_) age, _count ✅
Can contain letters, digits, underscore my_var2 ✅
Cannot start with a digit 2name ❌
Cannot use C keywords int, for ❌
Case sensitive Age ≠ age
No spaces allowed my var ❌

Types of Variables

Local Variable

Declared inside a function. Only accessible within that function.

Global Variable

Declared outside all functions. Accessible from any function in the program.

Static Variable

Retains its value between function calls. Declared with static keyword.

Extern Variable

Declared in another file. Used to share variables across multiple files.

Format Specifiers

Specifier Data Type Example
%d int printf("%d", 10);
%f float printf("%f", 3.14);
%lf double printf("%lf", 3.14);
%c char printf("%c", 'A');
%s string printf("%s", "hello");
